# Αναφορά ESP32 GPIO

## Ψευδώνυμα ακροδεκτών (Pin Aliases)

| ψευδώνυμο   | ακροδέκτης |
| :---        | :---       |
| builtin_led | 2          |
| red_led     | 2          |

## Κοινοί ακροδέκτες (ESP32 / ESP32-C3)

- **GPIO 2**: Ενσωματωμένη λυχνία LED σε πολλές πλακέτες ανάπτυξης (έξοδος).
- **GPIO 13**: Έξοδος γενικής χρήσης.
- **GPIO 21/20**: Συχνά χρησιμοποιούνται για UART0 TX/RX (αποφύγετε τη χρήση τους εάν χρησιμοποιείτε σειριακή επικοινωνία).

## Πρωτόκολλο (Protocol)

Ο κεντρικός υπολογιστής (host) του ZeroClaw αποστέλλει JSON μέσω σειριακής σύνδεσης (115200 baud):
- `gpio_read`: `{"id":"1","cmd":"gpio_read","args":{"pin":13}}`
- `gpio_write`: `{"id":"1","cmd":"gpio_write","args":{"pin":13,"value":1}}`

Απόκριση: `{"id":"1","ok":true,"result":"0"}` ή `{"id":"1","ok":true,"result":"done"}`
